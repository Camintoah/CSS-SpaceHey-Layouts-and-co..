# CSS,SpaceHey Layouts and co
Custom SpaceHey themes, CSS experiments, everything going into my mind..

Welcome to my little corner of the internet =)

This repository is mainly focused on creating and experimenting with CSS layouts for SpaceHey. Since SpaceHey no longer allows users to upload layouts directly to the official layout section, I use this repository as a place to store, share, and work on my themes.

## What you'll find here

- SpaceHey profile layouts
- CSS experiments
- Retro web-inspired designs
- Aero/ Old fashioned/ Game / fandom inspired themes
- Random coding projects and ideas

## About

While the main focus is learning and improving my CSS skills, I occasionally experiment with other types of coding and web development whenever an idea comes to mind.

Some projects may be unfinished, experimental, or simply created for fun and learning purposes.
